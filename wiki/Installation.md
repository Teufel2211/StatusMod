# Installation

1. Download the release JAR that matches your Fabric Minecraft version.
2. Place the JAR in your `mods/` folder.
3. Make sure Fabric Loader and Fabric API match the target Minecraft version.
4. Start or restart the game or server.

## Notes

- The shared Fabric line covers Minecraft `1.19` through `1.21.11`.
- The special Fabric 26.1 branch has its own release artifacts.
- StatusMod stores settings locally in `config/statusmod/`.
