# Compatibility

StatusMod is organized around two Fabric build paths:

- Shared Fabric: Minecraft `1.19` through `1.21.11`
- Dedicated Fabric 26.1: `26.1`, `26.1.1`, `26.1.2`

## Supported loader path

- Fabric only

## Not included in this wiki bundle

- Forge
- NeoForge
- Quilt
- Datapack

## Build inputs

- `Loader/fabric/`
- `Loader/fabric26.1/`
- `scripts/local/build-multiversion.ps1`
